## 🎮 操作方法 / Controls

本ゲームは **キーボード＆マウス** または **コントローラー** でプレイできます。  
**プレイ推奨：キーボード＆マウス**

⚠️ コントローラー操作はデバッグが不十分なため、一部不具合が発生する可能性があります。

- ゲーム終了：`Escape`

---

## ⌨️ キーボード＆マウス（推奨）

### メニュー操作
| 操作 | キー |
|---|---|
| 選択 | WASD |
| 決定 | Enter / マウス右クリック |

### ゲーム中操作
| 操作 | キー |
|---|---|
| 移動 | WASD |
| 攻撃 | マウス右クリック |
| ライト範囲 拡大 | Space（長押し） |
| ライト範囲 縮小 | Space（離す） |

### アイテム使用
| 手順 | キー |
|---|---|
| 構え | 左Shift（長押し） |
| 方向決め | WASD |
| 放つ | 左Shift（離す） |

### アイテム選択
| 手順 | キー |
|---|---|
| 開始 | E（長押し） |
| 選択 | A / D |
| 決定 | E（離す） |

### ポーズ
| 操作 | キー |
|---|---|
| 開始 | P |
| 選択 | WASD |
| 決定 | Enter / マウス右クリック |

---

## 🎮 コントローラー

### メニュー操作
| 操作 | ボタン |
|---|---|
| 選択 | 左スティック |
| 決定 | Bボタン |

### ゲーム中操作
| 操作 | ボタン |
|---|---|
| 移動 | 左スティック |
| 攻撃 | RT |
| ライト範囲 拡大 | LT（長押し） |
| ライト範囲 縮小 | LT（離す） |

### アイテム使用
| 手順 | ボタン |
|---|---|
| 構え | LB（長押し） |
| 方向決め | 左スティック |
| 放つ | LB（離す） |

### アイテム選択
| 手順 | ボタン |
|---|---|
| 開始 | D-Pad 上（長押し） |
| 選択 | RB / LB |
| 決定 | D-Pad 上（離す） |

### ポーズ
| 操作 | ボタン |
|---|---|
| 開始 | Select |
| 選択 | 左スティック |
| 決定 | Bボタン |
